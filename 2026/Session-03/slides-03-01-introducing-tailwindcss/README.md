# Welcome to Session 03

To install and update the slidedev code:

```shell
pnpm install
pnpm update
```

If you get errors when running the slides quit ( press <kbd>Q</kbd> **or** press <kbd>CTRL</kbd>+<kbd>C</kbd>) and
execute:

```shell
pnpm install slidev-theme-nmt
pnpm install slidev-addon-announcement
pnpm install shiki-magic-move shiki
pnpm install @iconify-json/fa7-brands @iconify-json/fa7-solid
pnpm update
```

To start the slide show:

```shell
pnpm run dev
```

- visit <http://localhost:3030>
